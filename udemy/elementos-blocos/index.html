Elementos Inline, Block e Inline Block em HTML

Elementos Inline

Os elementos inline em HTML são aqueles que não começam em uma nova linha e ocupam apenas a largura necessária. Exemplos de elementos inline incluem <span>, <a>, <img>, entre outros.

Elementos Block

Os elementos block em HTML são aqueles que ocupam toda a largura disponível da página, criando uma nova linha antes e depois do elemento. Exemplos de elementos block incluem <div>, <h1> a <h6>, <p>, <header>, <footer>, entre outros.

Elementos Inline Block

Os elementos inline block em HTML são aqueles que combinam características dos elementos inline e block. Eles não começam em uma nova linha, mas permitem definir largura e altura. Exemplos de elementos inline block incluem <img> (quando estilizado corretamente), <button>, entre outros.
