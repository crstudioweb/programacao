Trabalhando com classes e ids

Este é um exemplo de como trabalhar com classes e ids em HTML.

As classes e ids são atributos que podem ser adicionados a qualquer elemento HTML para aplicar estilos CSS ou manipulação JavaScript.

Por exemplo, você pode usar um id para identificar um elemento específico e uma classe para aplicar estilos a um grupo de elementos.

Exemplo de uso

Aqui está um exemplo de como usar classes e ids em HTML:

      <div id="meu-id">
        <p class="minha-classe">Este é um parágrafo com uma classe.</p>
      </div>

Neste exemplo, temos um elemento <div> com um id "meu-id" e um parágrafo dentro dele com a classe "minha-classe".

Você pode usar CSS para estilizar esses elementos da seguinte forma:

      #meu-id {
        background-color: lightblue;
      }

      .minha-classe {
        font-weight: bold;
      }

Conclusão

Neste exemplo, aprendemos como usar classes e ids em HTML para estilizar elementos com CSS. As classes permitem aplicar estilos a vários elementos, enquanto os ids são usados para identificar elementos específicos.

Divs e Spans

As <div> e <span> são elementos de bloco e em linha, respectivamente. O elemento <div> é usado para agrupar outros elementos e aplicar estilos a um bloco de conteúdo, enquanto o elemento <span> é usado para estilizar partes de um texto dentro de um bloco.

Em resumo, use <div> para agrupar elementos e aplicar estilos a blocos de conteúdo, e use <span> para estilizar partes específicas de um texto.

Bordas

As bordas podem ser estilizadas usando a propriedade CSS border. Você pode definir a largura, o estilo e a cor da borda. Aqui está um exemplo: 

        #meu-id {
          border: 2px solid red;
        }
        .minha-classe {
          border: 1px dashed blue;
        }

Fontes e Cores

As fontes e cores podem ser estilizadas usando as propriedades CSS font-family, font-size e color. Aqui está um exemplo: 

        #meu-id {
          font-family: Arial, sans-serif;
          font-size: 16px;
          color: #333;
        }
        .minha-classe {
          font-family: 'Courier New', monospace;
          font-size: 14px;
          color: #666;
        }
