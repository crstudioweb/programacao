Posicionamento em HTML

O que é posicionamento?

O posicionamento em HTML é uma técnica que permite controlar a disposição dos elementos na página.

Tipos de posicionamento

Os principais tipos de posicionamento em CSS são:

  • Posicionamento estático
  • Posicionamento relativo
  • Posicionamento absoluto
  • Posicionamento fixo
  • Posicionamento sticky

Exemplo de posicionamento

Veja abaixo um exemplo prático de como aplicar diferentes tipos de posicionamento em CSS: 

/* Exemplo de CSS para posicionamento */
.elemento-estatico {
    position: static;
}
.elemento-relativo {
    position: relative;
    top: 10px;
    left: 10px;
}
.elemento-absoluto {    
    position: absolute;
    top: 20px;
    left: 20px;
}
.elemento-fixo {
    position: fixed;
    top: 30px;
    left: 30px;
}
.elemento-sticky {
    position: sticky;
    top: 40px;
}
