Seletores

Seletores de Atributo

Os seletores de atributo permitem selecionar elementos com base em seus atributos e valores. Por exemplo: 

/* Seleciona todos os links que começam com "https" */
a[href^="https"] {
    color: green;
}

Este código seleciona todos os links que começam com "https" e os estiliza com a cor verde.

Seletores de Pseudo-Classe

Os seletores de pseudo-classe permitem selecionar elementos com base em seu estado ou posição. Por exemplo: 

/* Seleciona o primeiro parágrafo de cada seção */
section p:first-child {
    font-weight: bold;
}

Este código seleciona o primeiro parágrafo de cada seção e o estiliza com negrito.

Seletores de Pseudo-Elemento

Os seletores de pseudo-elemento permitem selecionar e estilizar partes específicas de um elemento. Por exemplo: 

/* Adiciona um conteúdo antes de cada parágrafo */
p::before {
    content: "Nota: ";
    font-weight: bold;      
}

Este código adiciona a palavra "Nota: " antes de cada parágrafo, estilizada em negrito.
